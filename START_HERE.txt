## 🚀 5-MINUTE START

```bash
# Terminal 1: Backend
cd backend
mvn clean install
mvn spring-boot:run

# Terminal 2: Frontend
cd frontend
npm install
npm run dev

# Browser: Open http://localhost:3000
```
